# Architecture Decision Records (mirror)

These files are the committed mirror of **accepted** ADRs. The authoritative records —
including drafts and superseded decisions — live in the engineering vault at
`vault4/04-Decisions-ADR/`. `[[wikilinks]]` inside these files resolve in the vault.

An ADR is copied here only once it reaches **Accepted** status.
